Latitude

Summary

______________________________________________________________

With the use of the OpenWeatherMap API, we gathered over 500 randomly selected countries to see how weather changes as you move closer to the equator. We used each country's latitude to see if it had any correlation to its respective max temperature, humidity, cloudiness, and wind speed as we approach the equator. Once we gathered all of the data we needed, we used Matplotlib to visualize our data.

This site provides all the data, plots, and analyis we came across.
